# Specification

## Goal

Create a functional AIEF starter experience.

## Requirements

- The README must explain AIEF simply.
- The repository must include universal AI assistant instructions.
- Assistant-specific files must reference `AGENTS.md` as the source of truth.
- The repository must include practical documentation.
- The repository must include templates for projects and changes.
- The repository must include at least one complete example.

## Acceptance Criteria

- [ ] `README.md` includes Quick Start, workflow, structure, and roadmap.
- [ ] `AGENTS.md` defines universal assistant behavior.
- [ ] `CLAUDE.md`, `GEMINI.md`, `CODEX.md`, and `CURSOR.md` exist.
- [ ] `docs/Getting-Started.md` exists.
- [ ] `docs/Workflow.md` exists.
- [ ] `templates/change/` contains `change.md`, `spec.md`, `tasks.md`, and `evidence.md`.
- [ ] `examples/todo-app/` contains one complete Change.

## Constraints

- Keep the starter simple.
- Avoid long theoretical documentation.
- Prefer Markdown.
- Do not require OpenSpec or Specboot for v1 starter usage.
