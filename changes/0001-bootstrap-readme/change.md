# Change

## ID

`0001-bootstrap-readme`

## Objective

Improve the initial AIEF repository experience so a new user can understand and start using the framework quickly.

## Scope

### In scope

- Improve `README.md`.
- Improve `AGENTS.md`.
- Add assistant-specific instruction files.
- Add getting started documentation.
- Add workflow documentation.
- Add starter templates.
- Add one complete example Change.

### Out of scope

- CLI tooling.
- Full OpenSpec integration.
- Full Specboot integration.
- Advanced enterprise compliance.

## Success Criteria

- A new user can understand AIEF from the README.
- A new user can create a Change using the templates.
- AI assistants have clear universal rules.
- Claude, Gemini, Codex, and Cursor have adapter files.
