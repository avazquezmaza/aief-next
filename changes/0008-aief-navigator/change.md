# Change

## ID
`0008-aief-navigator`

## Objective
Add AIEF Navigator: a guided entry point that helps users decide what to read, what to copy, what to install, what the AI uses, what to edit, and what not to touch.

## Scope

### In scope
- Add `docs/navigator/` as the guided entry point.
- Add decision tree documentation.
- Add new project path.
- Add existing project path.
- Add OS-specific installation guides for Windows, Linux, and macOS.
- Add AI assistant guide.
- Add tooling guide for AIEF, OpenSpec, and Specboot.
- Add workflow guide.
- Add Mermaid diagrams.

### Out of scope
- CLI changes.
- Automated installer.
- Website generation.
- Changes to current templates.
