# Change

## ID

`0005-v1-release-candidate`

## Objective

Complete the AIEF v1 MVP by adding agent profiles, Specboot adapter, practical developer documentation, and release readiness files.

## Scope

### In scope

- Add simple role-based agent profiles.
- Add lightweight Specboot adapter.
- Add practical docs for learning, migration, and workflow selection.
- Add GitHub community files.
- Add release notes for v1.0.
- Add basic GitHub Actions validation.

### Out of scope

- CLI implementation.
- Enterprise compliance.
- Full automation.
- Advanced multi-agent orchestration.

## Success Criteria

- AIEF v1 has enough structure to be usable.
- Users can understand how to start, how to use roles, and how to align with Specboot.
- The repository has basic open-source project files.
