# Change

## ID

`0004-openspec-adapter`

## Objective

Add a lightweight OpenSpec adapter that explains how AIEF Changes map to OpenSpec changes.

## Scope

### In scope

- Add `adapters/openspec/`.
- Document the conceptual mapping between AIEF and OpenSpec.
- Add minimal OpenSpec-compatible templates.
- Add one mapping example based on the Todo App change.

### Out of scope

- Installing OpenSpec.
- Requiring OpenSpec for AIEF usage.
- Replacing AIEF templates with OpenSpec templates.
- Adding CLI automation.

## Success Criteria

- Users understand that OpenSpec is optional.
- Users can map an AIEF Change to OpenSpec files.
- Users can use AIEF alone or with OpenSpec.
