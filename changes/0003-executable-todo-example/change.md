# Change

## ID

`0003-executable-todo-example`

## Objective

Turn the Todo App example into a small executable project that demonstrates AIEF with real code and tests.

## Scope

### In scope

- Add a minimal JavaScript Todo App example.
- Add executable source code.
- Add automated tests using Node.js built-in test runner.
- Add package scripts.
- Keep the example framework-free and easy to run.
- Keep the existing AIEF Change structure.

### Out of scope

- Web UI.
- Database persistence.
- Authentication.
- Frontend framework.
- Build tooling.

## Success Criteria

- A user can run the example with `npm test`.
- The example includes code, tests, and AIEF evidence.
- The example remains simple enough to understand in minutes.
