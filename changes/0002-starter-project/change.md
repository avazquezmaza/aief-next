# Change

## ID

`0002-starter-project`

## Objective

Add a copy-ready AIEF starter project that users can use as the base for new AI Engineering projects.

## Scope

### In scope

- Add `starter-project/`.
- Include universal and assistant-specific AI instruction files.
- Include minimal project documentation.
- Include `changes/`, `knowledge/`, `src/`, and `tests/`.
- Include one completed example Change inside the starter project.
- Include GitHub issue and PR templates for projects created from the starter.

### Out of scope

- CLI tooling.
- Real application implementation.
- Package manager configuration.
- OpenSpec or Specboot adapters.

## Success Criteria

- A user can copy `starter-project/` and immediately start a new AIEF project.
- The starter project explains where code, tests, changes, and knowledge go.
- The starter project includes one example completed Change.
