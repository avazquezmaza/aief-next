# Change

## ID
`0007-cli-mvp`

## Objective
Add a minimal AIEF CLI to automate common starter actions.

## Scope
- Add CLI package under `cli/`.
- Support `aief init`.
- Support `aief new-change <name>`.
- Support `aief use-profile <profile>`.
- Support `aief verify`.
- Support `aief release`.
- Add CLI documentation.

## Out of scope
- OpenSpec execution integration.
- Specboot execution integration.
- Publishing to npm.
- Advanced configuration.
