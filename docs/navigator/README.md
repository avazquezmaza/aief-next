## AIEF in One Sentence

AIEF coordinates the engineering workflow between humans, AI assistants, specifications, implementation, verification, and evidence.

## The Short Version

```text
Idea -> Change -> Spec -> Tasks -> Build -> Verify -> Evidence
```

## What AIEF Coordinates

```text
AIEF
  ├── OpenSpec   optional specification engine
  ├── Specboot   optional agent instruction bootstrap
  ├── AI tools   Claude, Gemini, Codex, Cursor, Copilot, ChatGPT
  └── Project    your actual source code and tests
```
