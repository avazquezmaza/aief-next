# Learning Path

Use this path if you are new to AIEF.

## 1. Read the README

Understand the basic idea:

```text
Idea -> Spec -> Tasks -> Build -> Verify -> Evidence
```

## 2. Open the Todo App example

```text
examples/todo-app/
```

Run:

```bash
npm test
```

## 3. Copy the starter project

```bash
cp -R starter-project my-project
```

## 4. Create your first Change

Create:

```text
changes/0001-my-first-change/
```

Add:

```text
change.md
spec.md
tasks.md
evidence.md
```

## 5. Work with an AI assistant

Give it:

```text
AGENTS.md
profiles/developer.md
changes/0001-my-first-change/
```

## 6. Capture evidence

Update `evidence.md`.

That is AIEF.
