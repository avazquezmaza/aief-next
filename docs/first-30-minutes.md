# Your First 30 Minutes

1. Clone the repository.
2. Read the README.
3. Run the Todo example:

```bash
cd examples/todo-app
npm test
```

4. Copy `starter-project/`.
5. Create `changes/0001-my-change/`.
6. Fill `change.md`, `spec.md`, `tasks.md`.
7. Give your AI assistant:

- AGENTS.md
- Optional profile
- Active Change

8. Implement.
9. Verify.
10. Update `evidence.md`.

Congratulations—you've completed the AIEF workflow.
