# Choosing Your Workflow

AIEF can be used at different levels.

## Small project

Use only:

```text
AGENTS.md
changes/
templates/change/
```

Best for:

- solo developers,
- experiments,
- small tools,
- documentation changes.

## Medium project

Add:

```text
starter-project/
profiles/
examples/
```

Best for:

- small teams,
- repeated feature work,
- AI-assisted implementation.

## Larger project

Add optional adapters:

```text
adapters/openspec/
adapters/specboot/
```

Best for:

- teams that need stronger governance,
- formal reviews,
- multiple AI assistants.

## Rule

Start with the smallest workflow that works.
