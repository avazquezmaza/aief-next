# Vision and Principles

## Vision

AIEF is a simple, tool-agnostic starter kit for AI Engineering.

It helps teams organize software work when humans and AI assistants collaborate.

## Mission

Make AI-assisted development consistent, traceable, and easy to adopt.

## Principles

### Human-Led

AI assists. Humans decide.

### Specification-Driven

A change should be understood before implementation begins.

### Simplicity First

The framework should be easier to use than to explain.

### Tool Agnostic

AIEF should work with any AI assistant or development environment.

### Evidence over Opinion

Work is complete when there is evidence that it works.

### Progressive Adoption

Start small. Add advanced practices only when needed.
