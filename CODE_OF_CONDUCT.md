# Code of Conduct

AIEF is intended to be a respectful and practical open-source project.

## Expected Behavior

- Be respectful.
- Assume good intent.
- Focus on ideas and evidence.
- Keep discussions constructive.

## Unacceptable Behavior

- Harassment.
- Personal attacks.
- Discrimination.
- Repeated disruptive behavior.

## Reporting

Open a private report with the repository maintainers.
