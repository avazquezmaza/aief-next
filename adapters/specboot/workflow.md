# SpecBoot Workflow with AIEF

## Simple workflow

```text
1. Choose active Change
2. Choose assistant
3. Choose optional profile
4. Provide context
5. Implement
6. Verify
7. Update evidence
```

## Example

```text
Use AGENTS.md.
Use CLAUDE.md.
Act as the Architect profile.
Review changes/0001-create-task/spec.md and suggest a simple design.
```

## Completion rule

A task is not complete until `evidence.md` is updated.
