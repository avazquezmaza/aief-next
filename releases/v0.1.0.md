# AIEF v0.1.0 Release Notes

## Theme

First usable AIEF Starter release.

## Included

- Lightweight AIEF workflow.
- Universal agent instructions.
- Assistant-specific instruction files.
- Starter project.
- Executable Todo App example.
- OpenSpec adapter.
- Specboot adapter.
- Agent profiles.
- AIEF Navigator.
- Practical documentation.
- Basic CLI MVP.
- GitHub templates.

## Not Included

- Published npm package.
- Enterprise compliance.
- Full multi-agent orchestration.
- Automated OpenSpec or Specboot validation.

## Recommended Audience

- Developers experimenting with AI-assisted engineering.
- Teams looking for simple specification-driven workflows.
- Architects evaluating human + AI collaboration patterns.
