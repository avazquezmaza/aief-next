# Project Agent Instructions

These instructions apply to all AI assistants working in this project.

## Prime Directive

AI assists. Humans decide.

## Rules

- Read the active Change before editing.
- Read `spec.md` before implementation.
- Read `tasks.md` before changing files.
- Keep changes small and focused.
- Do not modify unrelated files.
- Do not invent requirements.
- Ask when scope or requirements are ambiguous.
- Update documentation when behavior changes.
- Update `evidence.md` before considering the Change complete.

## Project Workflow

```text
Understand -> Plan -> Build -> Verify -> Document
```

## Completion Checklist

Before finishing, confirm:

- [ ] The active Change is clear.
- [ ] Requirements were implemented.
- [ ] Verification was performed.
- [ ] Evidence was updated.
- [ ] No unrelated changes were introduced.
