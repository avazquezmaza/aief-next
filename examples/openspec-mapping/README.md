# OpenSpec Mapping Example

This example shows how the AIEF Todo App Change can be represented in an OpenSpec-compatible structure.

Original AIEF Change:

```text
examples/todo-app/changes/0001-create-task/
```

OpenSpec-style mapping:

```text
examples/openspec-mapping/openspec/changes/0001-create-task/
```

Use this example only if your team wants OpenSpec-style change governance.
