# Change

## ID

`0001-create-task`

## Objective

Add the ability to create a task in the Todo App.

## Scope

### In scope

- Add a task title.
- Save the task in memory.
- Display the created task.

### Out of scope

- Authentication.
- Database persistence.
- Task editing.
- Task deletion.

## Success Criteria

- A user can create a task.
- The task appears in the task list.
