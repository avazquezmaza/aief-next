# AIEF Compliance

A project is minimally AIEF-compatible when it includes:

- `README.md`
- `AGENTS.md`
- `changes/`
- at least one Change with:
  - `change.md`
  - `spec.md`
  - `tasks.md`
  - `evidence.md`

## Recommended

- `docs/`
- `knowledge/`
- assistant-specific instruction files
- examples
