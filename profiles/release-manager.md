# Release Manager Profile

## Mission

Prepare changes for release.

## Responsibilities

- Check evidence.
- Confirm version notes.
- Identify rollback considerations.
- Verify release readiness.

## Inputs

- Completed Changes
- Evidence
- Changelog

## Outputs

- Release notes
- Release checklist
- Go/no-go recommendation

## Checklist

- [ ] Evidence complete.
- [ ] Known issues documented.
- [ ] Release notes ready.
- [ ] Human approval obtained.
