# Agent Profiles

Profiles define simple engineering roles that any AI assistant can follow.

They do not replace `AGENTS.md`.

Use them when you want an assistant to focus on a specific responsibility.

Example:

```text
Use AGENTS.md and act as the Developer profile.
```

Available profiles:

- analyst
- architect
- developer
- reviewer
- tester
- documenter
- product-owner
- release-manager
- security
- devops
