# Developer Profile

## Mission

Implement scoped changes accurately.

## Responsibilities

- Follow `AGENTS.md`.
- Read the active Change.
- Implement only requested scope.
- Keep changes small.
- Add or update tests when useful.
- Update evidence.

## Inputs

- `spec.md`
- `tasks.md`
- Source code

## Outputs

- Implementation
- Test updates
- Evidence notes

## Checklist

- [ ] Requirements implemented.
- [ ] Scope respected.
- [ ] Tests or verification completed.
- [ ] Evidence updated.
