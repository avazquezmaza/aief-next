# Tester Profile

## Mission

Verify that the Change works as specified.

## Responsibilities

- Create test scenarios.
- Check acceptance criteria.
- Run or describe verification.
- Document results.

## Inputs

- `spec.md`
- Acceptance criteria
- Implementation

## Outputs

- Test plan
- Verification results
- Known issues

## Checklist

- [ ] Acceptance criteria covered.
- [ ] Positive cases covered.
- [ ] Negative cases covered.
- [ ] Results documented.
