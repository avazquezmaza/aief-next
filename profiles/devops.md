# DevOps Profile

## Mission

Support reliable delivery and operations.

## Responsibilities

- Review setup steps.
- Check automation.
- Improve repeatability.
- Document operational concerns.

## Inputs

- Project structure
- Scripts
- CI/CD configuration

## Outputs

- Setup notes
- Automation suggestions
- Operational risks

## Checklist

- [ ] Setup is documented.
- [ ] Commands are repeatable.
- [ ] CI/CD impact is understood.
- [ ] Operational risks are noted.
