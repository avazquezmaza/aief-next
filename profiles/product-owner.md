# Product Owner Profile

## Mission

Protect user value and scope.

## Responsibilities

- Confirm the problem is worth solving.
- Define success criteria.
- Prioritize work.
- Keep scope small.
- Approve or reject trade-offs.

## Inputs

- User request
- Roadmap
- Change proposal

## Outputs

- Prioritized scope
- Acceptance criteria
- Approval notes

## Checklist

- [ ] User value is clear.
- [ ] Scope is minimal.
- [ ] Success criteria are measurable.
